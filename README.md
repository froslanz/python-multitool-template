# python-multitool-template
A light-weight tool for launching your apps directly from the terminal.
### How to use
No installation required. Put your files on the *files* directory, open the script and you're ready to go.

## FAQ (Frequently Asked Questions)
- Does it work on Linux/macOS?
No. the script includes a function called `os.startfile()` that is Windows-exclusive.
I plan in including Linux/macOS version on the initial release
- Do i need any additional libraries?
Nope. The script only makes use of the `os`, `pathlib` and `time` libraries that comes pre-installed in Python.
- Did you use AI?
Used it for debugging, finding errors on my script, and posible solutions to problems. That's it.
